In the modern NBA teams are using alot more "SMALL BALL" lineups. These are lineups without a traditional center. I set out to answer the question
are true centers becoming a non-factor, and being used less in the NBA or is this a myth. 

HOW TO RUN:
Put both files into same directory in any IDE and just run the code.
